Example programs using MorphAdorner facilities.

  1. AdornAString: Get part of speech tags, lemmata, and standard spellings for text in a string.
  2. PosTagString: Get part of speech tags for text in a string.
  3. SentenceAndTokenOffsets: Finding sentence and token offsets in text.
  4. UsingAnAdornedText: Using an adorned XML text.
