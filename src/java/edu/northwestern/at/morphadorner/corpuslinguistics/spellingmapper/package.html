BritishToUS is a simple filter which maps British spellings to American (US) spellings.

Usage: 

java -jar MorphTagger edu.northwestern.at.standardspelling.britishtous.BritishToUS <brit.txt >us.txt

where:

  • Standard input contains the British text to be mapped to American text. Use redirection (e.g., <brit.txt) to assign an arbitrary file for processing.

  • Standard output contains the Americanized text. Use redirection (e.g., >us.txt) to send the output to a file.

The British to American mapping word list comes from Kevin's Word List Page.
