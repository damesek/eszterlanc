Output generation for adorned text.

The outputter package contains classes for outputting text adorned by MorphAdorner.

All MorphAdorner outputters must implement the {@link edu.northwestern.at.morphadorner.corpuslinguistics.outputter.AdornedWordOutputter} interface. The {@link edu.northwestern.at.morphadorner.corpuslinguistics.outputter.AdornedWordOutputterFactory} provides the mechanism for instantiating a default or specified instance of an outputter implementation.
