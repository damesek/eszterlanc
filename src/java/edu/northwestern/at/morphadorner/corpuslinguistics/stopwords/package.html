Stop words.

All MorphAdorner stopword providers must implement the {@link edu.northwestern.at.morphadorner.corpuslinguistics.stopwords.StopWords} interface. The {@link edu.northwestern.at.morphadorner.corpuslinguistics.stopwords.StopWordsFactory} provides the mechanism for instantiating a default or specified instance of a stopwords implementation.

MorphAdorner includes two built-in stopword lists.

  • PorterStopWords, created by Martin Porter. This is the default MorphAdorner stop word list.
  • ChoiStopWords, created by Freddy Choi.
