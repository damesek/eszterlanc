Text Summarization.

Text Summarization attempts to produce a summary of the text in a document.

All MorphAdorner summarizers must implement the {@link edu.northwestern.at.morphadorner.corpuslinguistics.textsummarizer.TextSummarizer} interface. The {@link edu.northwestern.at.morphadorner.corpuslinguistics.textsummarizer.TextSummarizerFactory} provides the mechanism for instantiating a default or specified instance of a summarizer implementation.
